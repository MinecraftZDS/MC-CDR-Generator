# MC-CDR-Generator
A Minecraft command, datapack (under development), and resourcepack (under development) generator for Java Edition.

## (About Program)
In the title of the program, "C" means **"command"**, "D" means **"datapack"**, "R" means **"resourcepack"**.  
Alright, I have added more "normal" and "underworld" features (a Chinese meme describing abstract),  
so my application will be more interesting!  
Note: language is in `<root_dir>/lang`(same level as `<root_dir>/affiliated`)

## (Program Function)
The program currently only supports the generation of the `give` command and the `summon` command.  
With this software, commands such as the following can be generated:
```mcfunction
/give @s netherite_chestplate
/summon ~ ~ ~ armor_stand {Motion:[0.0, 0.5, -1.0]}
```
Of course, the development of datapacks and resourcepacks will also be carried out.  
Secondly, due to time constraints, the wiki link on the menu bar currently points to the Simplified Chinese interface. Please forgive us!  
The main body of the program window uses *Python's tkinter* architecture.  
The program is **still under development**. Please provide more support!
